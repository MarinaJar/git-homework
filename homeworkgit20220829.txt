Ветка (своими словами): отклонение от основной линии разработки для решения новых задач.
Есть основная ветка main, в которой хранится стабильная версия. 
Мы создаём ветки, решаем наши задачи (1 ветка - 1 задача), потом вливаем их в main.

создать: 	git branch testing
переключить: 	git checkout testing
просмотреть: 	git branch -v
удалить: 	git branch -D testing (удалится, если она не слита)
